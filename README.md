Nombre del proyecto: Fotografía perfecta
Slogan del proyecto: La mejor ruta para tus fotos
Como correrlo: Instalación de pip e instalar flask para poder correr el proyecto, y correrlo en el archivo de main.py
Stack Tecnológico: Flask, Leaflet, Tailwind.
Justificación del diseño: Se utilizo lso colores blanco, negro y amarillo para dar una impresión de elegancia y de un sistema premium, además de establecer prácticas útiles para la mejorar la accesibilidad del usuario.
Créditos a la IA: Este código fue co-creado con Gemini Canvas. Prompt principal: 'Crea una Landing Page HTML para una app de mapas llamada "fotografía perfecta". Debe tener un 'Hero' con una imagen de fondo de un mapa estilizado o topográfico, un título grande, y un botón CTA prominente que diga 'Explorar Mapa'. Usa Tailwind CSS. El diseño debe inspirar elegancia/limpieza, para demostrar una página donde fotógrafos profesionales o aficionados al área pueda encontrar las mejores rutas donde tomar fotografías.'
